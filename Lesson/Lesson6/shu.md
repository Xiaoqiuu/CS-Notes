# 1.1 树的简介

> 树(tree)由一个或一个以上的节点组成，且存在一个称为**树根**的特殊节点。

<img src = "C:\Archive\LanQiao\Lesson\Pictures\0008.png">

![](./Pictures/0008.png)

- **度数**: 每个节点所有子树的个数。
- **层数**: 树的层数。
- **高度**: 树的最大层数。
- **树叶**: 也称作**终端节点**，指*度数为0*的节点。

- 2,7的层数为2，1,4,6,8的层数为3。
- 5,2,7的度数为2，1,4,6,8的度数为0。
